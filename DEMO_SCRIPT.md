# Screen-recording voiceover script (~75s)

Warm the app first (open it, click Run analysis once) so nothing lags on camera.
Record at 1280×800+. Speak the **Say** lines; follow the **[Do]** cues.

---

**[Do] App open on the stadium map.**
**Say:** "This is MatchDay Ops Copilot — real-time decision support for a FIFA
World Cup 2026 stadium control room. Instead of just showing data, it makes a
call on what to do next."

**[Do] Click ▶ Go live. Let the numbers move for a moment.**
**Say:** "It runs live — crowd numbers climb, the map recolours, and every zone
projects how long until it's full. The board refreshes without calling the AI,
so it stays cheap."

**[Do] Point the cursor at a mid-density zone flagged high/elevated.**
**Say:** "And it's predictive, not reactive. This zone is only two-thirds full,
but it's flagged early — because it'll hit capacity in a few minutes. Density
alone would've missed that."

**[Do] Click Run analysis. Scroll to a recommendation.**
**Say:** "When I run the analysis, Gemini explains each risk — the reasoning,
the action, and a ready-to-read announcement in English, Spanish and French.
Notice the tone shifts: calm and reassuring for a medical incident, casual for a
busy restroom."

**[Do] Scroll to 'Ask the copilot'. Click "What if I close Gate B?"**
**Say:** "It's also conversational. I can ask a what-if — 'what if I close Gate
B?' — and it reasons from the live data: divert the flow, but not toward the gate
that's already critical with a medical incident."

**[Do] Scroll to Upload real data. Upload sample_data/zones.csv + incidents.csv.**
**Say:** "And it isn't hardcoded — evaluators can upload their own data and it
re-analyses instantly."

**[Do] Back to the top / map.**
**Say:** "Built with Gemini and FastAPI, fully tested, deployed live. Vibe coding
got me moving fast — thinking like an engineer made it production-ready. Thanks
for watching."

## On-screen captions (for muted autoplay)

Big, high-contrast text overlays — one per section, timed to the action. Keep
them short; viewers read at a glance.

| When | Caption overlay |
|---|---|
| Map appears | **Stadium ops — in real time** |
| Go live | **Live crowd flow · updates every 3s** |
| Predictive zone | **Flags zones BEFORE they fill** |
| Run analysis | **Explainable AI · EN · ES · FR** |
| Ask the copilot | **Ask it anything — even "what if?"** |
| Upload data | **Upload real data · nothing hardcoded** |
| Close | **Gemini + FastAPI · try it live 👇** |

Opening title card (first 1–2s): **MatchDay Ops Copilot — PromptWars Challenge 4**

## 30-second cut (punchy version)

**[Do] Map + Go live already running.**
**Say / caption:** "A live stadium control room — crowd flow updating in real time."

**[Do] Point at a zone flagged early.**
**Say / caption:** "It predicts which gates fill next, before they're full."

**[Do] Show a recommendation with es/fr.**
**Say / caption:** "Gemini explains each call — and writes the announcement in three languages."

**[Do] Click "What if I close Gate B?"**
**Say / caption:** "You can even ask it what-if questions."

**[Do] Cut to URL.**
**Say / caption:** "Built with Gemini + FastAPI. Live link below."

---

## Delivery tips
- Keep it under 90s — LinkedIn autoplays muted, so also add on-screen captions if
  you can.
- Speak a touch slower than feels natural; pause between sections.
- If a Gemini call is slow on camera, cut that few seconds in editing.
- End on the live URL or your name on screen.
